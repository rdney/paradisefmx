{% trans "2. Een reparatieverzoek indienen" %}

{% trans "Zo dient u een verzoek in:" %}

  1. {% blocktrans %} Klik op "Reparatie melden" op de homepage of in het menu. {% endblocktrans %}
  2. {% blocktrans %} Vul een duidelijke titel in (bijv. "Lamp kapot in consistorie"). {% endblocktrans %}
  3. {% blocktrans %} Beschrijf het probleem in de omschrijving. Hoe meer details, hoe beter. {% endblocktrans %}
  4. {% blocktrans %} Selecteer de locatie waar het probleem zich bevindt. {% endblocktrans %}
  5. {% blocktrans %} Kies de prioriteit: {% endblocktrans %}
    • {% trans "Laag" %} - {% trans "Kan wachten" %}
    • {% trans "Normaal" %} - {% trans "Binnen een week" %}
    • {% trans "Hoog" %} - {% trans "Zo snel mogelijk" %}
    • {% trans "Spoed" %} - {% trans "Direct actie nodig" %}
  6. {% blocktrans %} Vul uw naam in zodat we contact kunnen opnemen. {% endblocktrans %}
  7. {% blocktrans %} Voeg eventueel foto's toe om het probleem te verduidelijken. {% endblocktrans %}
  8. {% blocktrans %} Klik op "Indienen". {% endblocktrans %}
{% blocktrans %} Tip: U ontvangt een werkbonnummer. Bewaar dit nummer om de status te kunnen volgen. {% endblocktrans %}
